## 什么是 MCP？ 

MCP（模型上下文协议）是一个使大型语言模型（LLMs，如 Claude）能够与外部工具和数据源交互的协议。使用 MCP可以：

- 构建为 LLMs 提供工具和数据的服务器
- 将这些服务器连接到兼容 MCP 的客户端
- 通过自定义功能扩展 LLM 的能力

## 核心概念 

MCP 服务器可以提供三种主要类型的功能：

1. **工具**：可被 LLM 调用的函数（需要用户批准）
2. **资源**：可被客户端读取的类文件数据（如 API 响应或文件内容）
3. **提示词**：帮助用户完成特定任务的预设模板
